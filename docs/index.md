# PHP Ruleset Formatter — Documentation

This repository contains a small VS Code extension that formats PHP files using a lightweight interpretation of a PHPCS-style XML ruleset.

This site is a simple, in-repo documentation surface intended for quick reference. It contains copies of the editing and custom rules guides.

## Contents

- [Editing Rules](editing-rules.md) — how to change and test rule behavior, including the `else` cuddling rule.
- [Custom Rules](custom-rules.md) — guide for adding custom rules to the extension and contributing them back.

To publish these documents as a GitHub Pages site, set the Pages source in your repository settings to the `main` branch and the `/docs` folder.
